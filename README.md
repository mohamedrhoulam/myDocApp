## Test the web app 
- Git clone the repo
  ```bash
  git clone https://github.com/mohamedrhoulam/myDocApp.git
  ```
- Install dependencies
    ```bash
    npm install
    ```
- Create a new database and run the sql scripts found in the server direcory
- run the server and the front end.
